    ==>> description \\
    \begin{description}
        \item[引擎] 全称为排版引擎，是编译源代码并生成文档的程序，如 \hologo{pdfTeX}、\hologo{XeTeX} 等。有时也称为编译器。
        \item[格式] 是定义了一组命令的代码集。\LaTeX{} 就是最广泛应用的一个格式，高德纳本人还编写了一个简单的 \hologo{plainTeX} 格式，
        没有定义诸如 \cmd{document\-class} 和 \cmd{section} 等等命令。
    \end{description}
